# Student Record Management System

A simple Python project that manages student records using Object-Oriented Programming and file handling.

## Features

* Add student
* View students
* Delete student
* Update marks

## Concepts Used

* Python OOP (Classes & Objects)
* File Handling
* Lists

## Data Storage

Student data is stored in a text file called `students.txt`.

## How to Run

1. Make sure Python is installed.
2. Run the program:

```
python main.py
```
